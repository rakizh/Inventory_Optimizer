# Inventory Optimizer

A full-stack inventory management system with AI-powered recommendations and Zoho Books integration.

## Features
- Real-time inventory metrics dashboard
- AI-powered dead stock & slow-moving item detection
- Automatic reorder point calculation
- Zoho Books/Inventory API integration
- REST API with comprehensive endpoints
- React TypeScript frontend

## Tech Stack
- **Backend:** Python, FastAPI, SQLAlchemy
- **Frontend:** React, TypeScript, Vite
- **Database:** PostgreSQL
- **Integration:** Zoho Books API
- **Testing:** pytest
- **Deployment:** Docker, CI/CD (GitHub Actions)
